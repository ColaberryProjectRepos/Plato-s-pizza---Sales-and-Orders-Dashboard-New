Creating Parameter and Filter 8



Parameter: 


1. Under Data pane, Click on the down arrow and select the option Create Parameter.

2. Name – Pick Date

 Date Type – Date

 Current Value – 12/31/2015

 Allowable Values – Range 

 Minimum – 1/1/2015

 Maximum – 12/31/2015

 Step Size – 1 Day

3. Click OK.


Filter:

1. Right click on Pick Date under Parameters tab and select Create Calculated Field.

 Name = Date Filter 

 [Order Date] <= [Pick Date]

Click OK.
